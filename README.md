# Direct Conversion Receiver
This project is a 7MHz Direct Conversion Reciever intended for learning and research purposes to better understand the design and validation of RF and radio systems. 

## Roadmap

### 1. LTSpice / PySpice
Full simulation and validation of:  
- [ ] Diode Ring Mixer
- [x] Band-Pass Filter (BPF)
- [x] Colpitts Oscillator
- [ ] 3-Stage Audio Amplifier

### 2. KiCad
Each Stage will be created as a PCB to verify each section both seperately and alongside the rest of the PCB. 
- [ ] Schematic
- [ ] PCB Layout

### 3. Bench Testing
Validation tests will be performed on each Stage in reference to LTSpice and hand calculations. 
- [ ] Validate Diode Ring Mixer
- [ ] Validate Colpitts Oscillator
- [ ] Validate BPF
- [ ] Validate 3-Stage Audio Amplifier
- [ ] Validate DCR
